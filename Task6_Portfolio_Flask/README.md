# Portfolio Website with Flask

## Description
This project is a personal portfolio website developed using Flask, HTML, and CSS as part of Python Developer Internship Task 6.

The website contains personal information, skills, projects, and a contact form.

## Features
- Personal portfolio homepage
- About Me section
- Skills section
- Projects section
- Contact form
- Flask routing
- HTML template rendering
- CSS styling

## Technologies Used
- Python
- Flask
- HTML
- CSS
- VS Code

## Concepts Used
- Flask Routing
- Templates
- Jinja2
- HTML Forms
- Static Files
- CSS Styling

## Installation

Install Flask:

pip install flask

## How to Run

Run the app:

python app.py

Open in browser:

http://127.0.0.1:5000
