# Portfolio Website with Flask

## Description

This project is a personal portfolio website developed using Flask, HTML, and CSS as part of Python Developer Internship Task 6.

The website showcases my profile, skills, projects, education, and contact information through a simple and responsive web interface.

---

## Features

* Personal portfolio homepage
* About Me section
* Skills section
* Projects section
* Education section
* Contact form
* Flask routing
* HTML templates
* CSS styling
